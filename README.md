StayVora
StayVora is a modern web application that helps users find, explore, and book vacation rentals with ease. Whether it’s a beach house, mountain retreat, or city apartment, StayVora provides a seamless experience for travelers to discover their perfect stay.

Features
User Authentication: Sign up and log in securely.Property Listings:
Browse various types of properties with images, descriptions, and prices.
Responsive Design: Works flawlessly on desktop, tablet, and mobile devices.
Booking Management: Users can view and manage their bookings.
Admin Panel (Optional): Manage listings, users, and bookings.

Tech Stack
Frontend: HTML, CSS, JavaScript, EJS
Backend: Node.js, Express.js
Database: MongoDB
Deployment: Render / Vercel
Other: REST APIs for dynamic data handling
